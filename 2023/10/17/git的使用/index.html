git 基本使用

git 分为 3 个区

  • 工作区 —— 存放文件
  • 暂存区 —— 工作区变更，提交到暂存区
  • 版本库 —— 暂存区的内容，提交到版本库

status 颜色

  • 红: 工作区变更了,但是没有提交到暂存区,此时 git checkout . 会撤回变更无法找回
  • 绿: 已经提交到暂存区
  • 无: 都被 git 管理起来了

空文件夹不会被 git 管理

git 忽略文件

  1. 根目录下新建.gitignore文件
  2. 在里面写需要忽略的文件
